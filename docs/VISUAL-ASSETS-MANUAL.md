# BADHUCHE - VISUAL ASSETS MANUAL

**Document Purpose:** This lists every texture, gradient, pattern, and visual asset that should be sourced from stock libraries or created in Photoshop. These elements are critical for the premium feel and should NOT be generated by AI.

## 1. BACKGROUND TEXTURES

### 1.1 Noise/Grain Texture
| Property | Specification |
|----------|---------------|
| **Purpose** | Subtle overlay on all pages for depth |
| **Type** | Fine film grain / noise |
| **Opacity when used** | 2-4% |
| **Format** | PNG with transparency |
| **Size** | 500x500px (tileable) |
| **Color** | Monochrome (white noise on transparent) |
| **Where to source** | [Textures.com](https://www.textures.com), [Subtle Patterns](https://www.toptal.com/designers/subtlepatterns/) |
| **Photoshop method** | Filter > Noise > Add Noise (Gaussian, 3-5%, Monochromatic) on white layer, then set to Screen blend |

**Usage Locations:**
- Full page background overlay
- Hero sections
- Glass card backgrounds
- Footer background

## 2. GRADIENT ASSETS

### 2.1 Hero Gradient (Primary)
| Property | Specification |
|----------|---------------|
| **Purpose** | Main background gradient for hero and dark sections |
| **Type** | Multi-stop radial/linear gradient |
| **Format** | CSS values + Photoshop PSD backup |
| **Colors** | See below |

**Gradient Stops:**
\`\`\`
Stop 1: #0A0F1C (Deep Navy) - 0%
Stop 2: #141820 (Dark Surface) - 40%
Stop 3: #1A1F2E (Elevated Surface) - 70%
Stop 4: #0D1117 (Near Black) - 100%
\`\`\`

**Photoshop Creation:**
1. Create 1920x1080px canvas
2. Fill with #0A0F1C
3. Add radial gradient overlay from center-top
4. Blend #1E3A5F at 20% opacity in top-right
5. Add subtle #2D1B4E at 10% opacity in bottom-left
6. Export as JPG for fallback

## 6. PHOTOGRAPHY TREATMENT PRESETS

### 6.1 Project Image Treatment
All project photos should have consistent treatment:

**Photoshop Action Steps:**
1. Slight desaturation: -10 to -15 vibrance
2. Lift blacks: Curves adjustment, lift bottom-left point
3. Add warmth: Photo Filter > Warming (85) at 10%
4. Subtle vignette: -15 to -20
5. Sharpen for web: Smart Sharpen, Amount 50%, Radius 0.5px

**Export Settings:**
- Format: JPG
- Quality: 80-85%
- Color Profile: sRGB
- Resolution: 72dpi

---

### 6.2 Team Photo Treatment
**Photoshop Action Steps:**
1. Convert to B&W (for consistency)
2. Add slight sepia tone: Hue/Sat adjustment layer, Colorize, Hue 35, Sat 10
3. Soft contrast boost: Curves S-curve
4. Subtle grain: Add Noise 1-2%
